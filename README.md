# PSMAPET_radiomics
The R file in the repository can be used as a template to select predictors using unsupervised feature selection (variance thresholding).
The Python script in the repository can be used as a template to perform imbalance correction on train data, train the cox regression model on train data using selected predictors and fit the model on the test data
The MATLAB script in the repository can be used as a template to perform five-year MFS classification on train data (imbalance corrected) using selected predictors and fit the model on test data
